## 1. Get your API key

Sign up at [app.answira.com](https://app.answira.com/register), then go to **Settings → API Keys** and create a new key with the scopes you need.

```bash
# Your API key looks like this
ak_live_abc123def456...
```

## 2. Create a chatbot

<CodeGroup>
```bash cURL
curl -X POST https://api.answira.com/v1/chatbots \
  -H "X-API-Key: ak_live_YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Support Bot",
    "description": "Customer support for Acme Inc",
    "system_prompt": "You are a helpful support agent for Acme Inc. Be friendly and concise.",
    "ai_model": "anthropic/claude-sonnet-4-5",
    "bot_type": "custom",
    "temperature": 0.7,
    "max_tokens": 500
  }'
```

```python Python
import requests

response = requests.post(
    "https://api.answira.com/v1/chatbots",
    headers={"X-API-Key": "ak_live_YOUR_KEY"},
    json={
        "name": "Support Bot",
        "description": "Customer support for Acme Inc",
        "system_prompt": "You are a helpful support agent for Acme Inc.",
        "ai_model": "anthropic/claude-sonnet-4-5",
        "bot_type": "custom",
        "temperature": 0.7,
    }
)
chatbot = response.json()
print(f"Created: {chatbot['id']}")
```

```javascript JavaScript

```
</CodeGroup>

## 3. Train it with your data

Upload training data so your bot can answer questions about your product.

<CodeGroup>
```bash cURL
# Train from text
curl -X POST https://api.answira.com/v1/chatbots/{chatbot_id}/training/text \
  -H "X-API-Key: ak_live_YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Pricing FAQ",
    "content": "Q: How much does it cost?\nA: Our starter plan is $29/month with 1,000 messages included.\n\nQ: Is there a free trial?\nA: Yes, we offer a 14-day free trial with full access."
  }'

# Train from URLs
curl -X POST https://api.answira.com/v1/chatbots/{chatbot_id}/training/urls \
  -H "X-API-Key: ak_live_YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "urls": ["https://acme.com/docs", "https://acme.com/faq"],
    "mode": "individual"
  }'
```
</CodeGroup>

<Note>
  Training is asynchronous. Use the [training status endpoint](/api-reference/training-status) to check progress.
</Note>

## 4. Chat with your bot

<CodeGroup>
```bash cURL
# Start a session
SESSION=$(curl -s -X POST https://api.answira.com/v1/sessions \
  -H "Content-Type: application/json" \
  -d '{"chatbot_id": "YOUR_CHATBOT_ID"}' | jq -r '.session_id')

# Send a message (streaming)
curl -N -X POST https://api.answira.com/v1/messages/stream \
  -H "Content-Type: application/json" \
  -d "{\"session_id\": \"$SESSION\", \"message\": \"What are your pricing plans?\"}"
```

```python Python
import requests

# Start session
session = requests.post(
    "https://api.answira.com/v1/sessions",
    json={"chatbot_id": "YOUR_CHATBOT_ID"}
).json()

# Send message
response = requests.post(
    "https://api.answira.com/v1/messages",
    json={
        "session_id": session["session_id"],
        "message": "What are your pricing plans?"
    }
).json()

print(response["response"])
```
</CodeGroup>

## 5. Embed the widget

Add this snippet to any HTML page:

```html
<script src="https://api.answira.com/static/premium-widget-v4.js"></script>
<script>
  BGAIPremiumWidget.init({
    chatbotId: 'YOUR_CHATBOT_ID',
    apiUrl: 'https://api.answira.com',
    position: 'bottom-right',
    primaryColor: '#3d8b7a',
    widgetTitle: 'Support Bot',
    greeting: 'Hi! How can I help you today?'
  });
</script>
```

That's it — your AI chatbot is live. Visit the [Widget Customization](/widget/customization) guide to style it to match your brand.
