The BGAI chat widget is a drop-in JavaScript component that adds an AI-powered chat interface to any website. It connects to your chatbot and handles sessions, streaming, handoff, and visitor tracking out of the box.

## Features

- Real-time streaming responses
- Light and dark themes
- Fully customizable colors, fonts, and positioning
- Visitor session persistence (conversations resume across pages)
- Human handoff support
- Message reactions (thumbs up/down)
- UTM parameter tracking
- HMAC visitor identity verification
- Responsive design (desktop and mobile)
- Shadow DOM isolation (no CSS conflicts)

## Quick start

```html
<script src="https://api.answira.com/static/premium-widget-v4.js"></script>
<script>
  BGAIPremiumWidget.init({
    chatbotId: 'YOUR_CHATBOT_ID',
    apiUrl: 'https://api.answira.com'
  });
</script>
```
