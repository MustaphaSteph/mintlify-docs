# Welcome to BGAI

BGAI (Answira AI) is an enterprise-ready AI chatbot platform that lets you create intelligent support agents, train them on your data, and embed them anywhere with a single script tag.

## Key features

- **Vector search** — Semantic retrieval from your training data using embeddings
- **Streaming responses** — Real-time SSE streaming for instant token-by-token output
- **Multi-model** — Choose from OpenAI, Anthropic Claude, Google Gemini, and more
- **Team management** — Invite agents, assign skills, route conversations intelligently
- **Analytics** — Track conversations, response quality, handoff rates, and agent performance
- **Webhooks** — Get notified on conversations, handoffs, leads, and training events
- **API-first** — Full REST API with scoped API keys for programmatic access

## Architecture overview

```
Your Website → Chat Widget → BGAI API → AI Model (OpenAI/Claude/Gemini)
                                ↓
                        Vector Search (Qdrant)
                                ↓
                        Knowledge Base (PostgreSQL)
                                ↓
                        Agent Dashboard (React)
```
