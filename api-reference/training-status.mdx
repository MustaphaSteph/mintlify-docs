---
title: Training Status
api: GET /v1/chatbots/{chatbot_id}/training/status
description: Check the status of training data processing
---

Requires `training:read` scope.

<ParamField path="chatbot_id" type="string" required>
  The chatbot ID.
</ParamField>

<RequestExample>
```bash cURL
curl https://api.answira.com/v1/chatbots/bot_abc123/training/status \
  -H "X-API-Key: ak_live_YOUR_KEY"
```
</RequestExample>

<ResponseExample>

  "is_active": true,
  "files": [
