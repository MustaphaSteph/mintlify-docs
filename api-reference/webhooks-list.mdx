---
title: List Webhooks
api: GET /v1/chatbots/{chatbot_id}/webhooks
description: List all webhook subscriptions for a chatbot
---

Requires `webhooks:read` scope.

<ParamField path="chatbot_id" type="string" required>
  The chatbot ID.
</ParamField>

<RequestExample>
```bash cURL
curl https://api.answira.com/v1/chatbots/bot_abc123/webhooks \
  -H "X-API-Key: ak_live_YOUR_KEY"
```
</RequestExample>

<ResponseExample>
```json 200
{
  "webhooks": [
    {
      "id": "wh_abc123",
      "url": "https://api.yourapp.com/webhooks/bgai",
      "events": ["conversation.started", "handoff.triggered"],
      "created_at": "2025-01-15T10:00:00Z"
    }
  ]
}
```
</ResponseExample>
