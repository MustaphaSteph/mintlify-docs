---
title: End Session
api: POST /v1/sessions/{session_id}/end
description: End an active chat session
---

<ParamField path="session_id" type="string" required>
  The session ID.
</ParamField>

<RequestExample>
```bash cURL
curl -X POST https://api.answira.com/v1/sessions/ses_1234567890/end
```
</RequestExample>

<ResponseExample>
```json 200
{
  "status": "ended",
  "session_id": "ses_1234567890"
}
```
</ResponseExample>
