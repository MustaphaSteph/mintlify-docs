---
title: Add Reaction
api: POST /v1/messages/reactions
description: Add a thumbs up/down reaction to a message
---

<ParamField body="message_id" type="string" required>
  The message ID to react to.
</ParamField>

<ParamField body="reaction" type="string" required>
  The reaction type: `thumbs_up` or `thumbs_down`.
</ParamField>

<ParamField body="session_id" type="string" required>
  The session ID the message belongs to.
</ParamField>

<RequestExample>
```bash cURL
curl -X POST https://api.answira.com/v1/messages/reactions \
  -H "Content-Type: application/json" \
  -d '{
    "message_id": "msg_abc123",
    "reaction": "thumbs_up",
    "session_id": "ses_1234567890"
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "status": "ok"
}
```
</ResponseExample>
