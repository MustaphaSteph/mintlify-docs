---
title: List Messages
api: GET /v1/sessions/{session_id}/messages
description: Fetch all messages in a session
---

<ParamField path="session_id" type="string" required>
  The session ID.
</ParamField>

<ParamField query="since" type="string">
  ISO 8601 timestamp. Only return messages after this time.
</ParamField>

<ParamField query="limit" type="integer" default="50">
  Maximum number of messages to return.
</ParamField>

<RequestExample>
```bash cURL
curl "https://api.answira.com/v1/sessions/ses_1234567890/messages?limit=20"
```
</RequestExample>
