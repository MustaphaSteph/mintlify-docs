---
title: Get Chatbot
api: GET /v1/chatbots/{chatbot_id}
description: Get detailed chatbot configuration
---

Requires `chatbots:read` scope.

<ParamField path="chatbot_id" type="string" required>
  The chatbot ID.
</ParamField>

<RequestExample>
```bash cURL
curl https://api.answira.com/v1/chatbots/bot_abc123 \
  -H "X-API-Key: ak_live_YOUR_KEY"
```
</RequestExample>

<ResponseExample>

  "name": "Support Bot",
  "description": "Customer support agent",
  "system_prompt": "You are a helpful support agent...",
  "bot_type": "custom",
