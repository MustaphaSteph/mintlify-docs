---
title: Delete Webhook
api: DELETE /v1/chatbots/{chatbot_id}/webhooks/{webhook_id}
description: Remove a webhook subscription
---

Requires `webhooks:write` scope.

<ParamField path="chatbot_id" type="string" required>The chatbot ID.</ParamField>
<ParamField path="webhook_id" type="string" required>The webhook ID.</ParamField>

<RequestExample>
```bash cURL
curl -X DELETE https://api.answira.com/v1/chatbots/bot_abc123/webhooks/wh_abc123 \
  -H "X-API-Key: ak_live_YOUR_KEY"
```
</RequestExample>

<ResponseExample>
```json 200
{"status": "deleted"}
```
</ResponseExample>
